1. Memulai (Getting Started)1. Getting Started

VEX Anti-Cheat menggunakan arsitektur Hybrid Cloud Security yang menjembatani antara Roblox Engine, Backend API, dan Database Relasional untuk mitigasi ancaman secara real-time. VEX Anti-Cheat utilizes a Hybrid Cloud Security architecture bridging the Roblox Engine, Backend API, and Relational Database for real-time threat mitigation.

1.1 Alur Instalasi Paket .rbxm1.1 Installation Workflow

Ikuti prosedur standar instalasi paket distribusi:Follow the standard installation package procedure:

  • Step 1: Akses server Discord resmi dan gunakan perintah /create <PlaceId> untuk mendapatkan Secret Key.Access the official Discord server and use the command /create <PlaceId> to obtain the Secret Key.
  • Step 2: Unduh berkas VEX_AntiCheat_v2.rbxm.Download the VEX_AntiCheat_v2.rbxm file.
  • Step 3: Masuk ke Roblox Studio > Explorer > Insert from File.Go to Roblox Studio > Explorer > Insert from File.
  • Step 4: Distribusikan struktur folder ke Service yang sesuai:Distribute folder structure to appropriate Services:
    • VEX (Folder) ➔ ServerScriptService
    • VEX_TopbarIcon ➔ ReplicatedStorage
    • VEX_AdminPanel ➔ StarterGui
    • VEX_ClientHandler & VEX_ExecutorScanner ➔ StarterPlayerScripts

1.2 Aktivasi & Konfigurasi1.2 Activation & Configuration

Aktifkan Allow HTTP Requests pada Game Settings > Security. Konfigurasi VEX_Config.lua seperti berikut:Enable Allow HTTP Requests in Game Settings > Security. Configure VEX_Config.lua as follows:

Config.SECRET_KEY = "VEX-XXXX-XXXX"
Config.ADMIN_IDS  = { 12387945654, 12345678 }

⚠️ Catatan: Berkas VEX_BackendURL.lua telah terenkripsi. Dilarang keras memodifikasi isinya.Note: The VEX_BackendURL.lua file is encrypted. Strictly forbidden to modify its contents.

2. Core Security Logic2. Core Security Logic

2.1 Client-Side VM Scanner2.1 Client-Side VM Scanner

Mendeteksi injeksi executor melalui pemindaian berkala pada fungsi global environment dan hook detection.Detects executor injection through periodic scanning of global environment functions and hook detection.

2.2 Server Behavioral Radar2.2 Server Behavioral Radar

Validasi posisi horizontal dan vertikal via koordinat fisika server (AssemblyLinearVelocity) untuk mencegah manipulasi posisi klien (anti-spoof).Validates horizontal and vertical position via server physics coordinates (AssemblyLinearVelocity) to prevent client position manipulation (anti-spoof).

2.3 Proteksi Aset (Poison Pill)2.3 Asset Protection (Poison Pill)

Mengunci aset menggunakan tanda tangan digital. Jika map dicuri paksa menggunakan ekspor, sistem akan memicu mekanisme *Poison Pill* yang merusak integritas koordinat part dan mengirim sinyal audit ke pemilik.Locks assets using digital signatures. If the map is forcibly stolen via export, the system triggers a *Poison Pill* mechanism that corrupts part coordinate integrity and sends an audit signal to the owner.
